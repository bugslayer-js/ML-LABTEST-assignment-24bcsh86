# Numeric & categorical feature lists
numeric_features = ["annual_income", "debt_to_income_ratio",
                    "credit_score", "loan_amount", "interest_rate"]

categorical_features = ["gender", "marital_status", "education_level",
                        "employment_status", "loan_purpose", "grade_subgrade"]

# Numeric pipeline: impute + robust scaling (to reduce outlier impact)
numeric_transformer = Pipeline(steps=[
    ("imputer", SimpleImputer(strategy="median")),   # handles missing numerics
    ("scaler", RobustScaler())                       # robust to outliers
])

# Categorical pipeline: impute + one-hot encode
categorical_transformer = Pipeline(steps=[
    ("imputer", SimpleImputer(strategy="most_frequent")),  # handles missing categoricals
    ("onehot", OneHotEncoder(handle_unknown="ignore", sparse_output=False))
])

# Full preprocessing transformer
preprocessor = ColumnTransformer(
    transformers=[
        ("num", numeric_transformer, numeric_features),
        ("cat", categorical_transformer, categorical_features)
    ]
)

# Section 3: Model Development & Evaluation

# Create Logistic Regression model
model = LogisticRegression(max_iter=500, solver="lbfgs")

# Final pipeline combining preprocessing + model
clf = Pipeline(steps=[
    ("preprocessor", preprocessor),
    ("model", model)
])

# Train the model
clf.fit(X_train, y_train)

# Predict probabilities for the positive class (loan_paid_back = 1)
y_proba = clf.predict_proba(X_test)[:, 1]

# Compute ROC AUC
overall_auc = roc_auc_score(y_test, y_proba)
print(f"Overall Test ROC AUC: {overall_auc:.4f}")

** Conclusion & Recommendations **

The goal of this project was to build a predictive system that helps FinSecure assess the risk of a loan default at the time of application. Using historical loan data, we developed a machine learning model that estimates the probability of a loan being successfully paid back.

The final Logistic Regression model achieved a ROC AUC score of 0.9119 on the test set. This indicates that the model is highly effective at distinguishing between applicants who are likely to repay and those who are at higher default risk. This performance represents a significant improvement over the existing manual scorecard approach, which lacks the ability to capture complex relationships in the data.

A fairness assessment was also conducted across key customer subgroups:

Education Levels: AUC ranged from approximately 0.908 to 0.925 No group displayed unusual bias or risk of systematic disadvantage.

Loan Purposes: AUC ranged from approximately 0.901 to 0.923 Performance variations were reasonable and did not indicate discrimination against any category.

Overall, the model demonstrates both strong predictive ability and fair treatment of different borrower groups.

Project Summary

This project focuses on building a predictive model for FinSecure to assess the likelihood of a borrower defaulting on a loan. The work was structured into four key sections:

Section 1: Problem Formulation & Target Variable Analysis

We defined the business objective: Predict loan default risk to support faster and data-driven credit decisions.

The target variable loan_paid_back was analyzed:

1 = loan fully repaid

0 = loan defaulted

Data distribution shows:

~20% defaults (moderate class imbalance)

Understanding this helped us choose ROC AUC as the primary evaluation metric since it is threshold-independent and ideal for imbalanced classification.

Section 2: Feature Engineering & Preprocessing Pipeline

We prepared the dataset to ensure high-quality model inputs:

Checked for missing values none present, but still handled in pipeline using SimpleImputer

Examined numeric features for outliers and applied RobustScaler to reduce their impact

One-Hot Encoding was used to convert all categorical variables like education_level and loan_purpose

Removed non-informative ID column

Created a train-test split (80% / 20%) with stratify to preserve class balance

Combined transformations into a single scalable preprocessing pipeline using ColumnTransformer

This ensured clean, properly scaled, machine-learning-ready data.

Section 3: Model Development and Tuning

Built a Logistic Regression model wrapped in a pipeline

Trained on the processed training data

Generated probability predictions for test loans

Evaluated using ROC AUC

Achieved 0.9119, indicating strong risk differentiation between good and bad loans

Plotted and interpreted the ROC Curve, confirming excellent predictive power

This model significantly improves over manual scoring and is suitable for deployment.

Section 4: Subgroup Fairness Analysis

To ensure ethical and unbiased decisions:

Calculated performance by education level

AUC values ranged from approx. 0.908 to 0.925

Calculated performance by loan purpose

AUC ranged from approx. 0.901 (Car loans) to 0.923 (Education loans)

All subgroups performed well with no indication of discrimination, demonstrating fair model behavior across borrower categories.

Overall Outcome

Goal Result Predict loan default accurately ROC AUC = 0.912 Automate & enhance credit decisioning Strong model performance Ensure fairness across groups Minimal variation in subgroup AUC

Final Recommendation

The model is deployment-ready. It can support underwriting decisions, interest rate pricing, and risk monitoring while maintaining fairness and transparency.

Future enhancements could include:

Advanced boosting models (XGBoost / HGBClassifier)

SHAP-based explainability
